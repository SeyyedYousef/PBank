# PBank Documentation Index

## Core Documentation (مستندات اصلی)
این فایل‌ها مستندات اصلی و جاری پروژه هستند که برای پیاده‌سازی استفاده می‌شوند.

*   **[Master_Vision.md](Master_Vision.md):** اهداف کلی پروژه، مخاطبان هدف، و خط قرمزهای امنیتی (Vision).
*   **[Tech_Stack.md](Tech_Stack.md):** استک فنی، زبان‌ها، قوانین دیتابیس و پکیج‌ها (Architecture).
*   **[UI_UX_Rules.md](UI_UX_Rules.md):** قوانین دیزاین، پالت رنگی، تایپوگرافی و رفتار کامپوننت‌ها (Design System).
*   **[Business_Logic.md](Business_Logic.md):** منطق تجاری، فرمول‌های محاسبه کارمزد، و فلوچارت‌های تراکنش (Logic).
