# PBank Section: Profile (تنظیمات & آمار)
> **Layout:** Stats -> Settings -> Security
> **Goal:** User Control

پروفایل کاربر، هم داشبورد مدیریتی است و هم مرکز امنیت.

## 1. آمار و وضعیت (Stats & Status)
*   نمایش گرافیکی و جذاب وضعیت کاربر.
*   "شما در این ماه **۱۵۰۰۰ افغانی** تراکنش داشتید".
*   نمایش سطح کاربری (مثلاً "کاربر نقره‌ای").

## 2. امنیت (Security Settings)
*   **پین کد (PIN Code):** تغییر رمز ورود.
*   **بیومتریک:** فعال‌سازی اثر انگشت/فیس‌آیدی.
*   **مدیریت دستگاه‌ها:** مشاهده دستگاه‌های متصل.

## 3. تنظیمات عمومی
*   زبان (دری/پشتو/انگلیسی).
*   تم (Dark/Light).
*   پشتیبانی.

## 4. معماری (Microservices Consideration)
*   **Identity Service:** مسئول تمام تنظیمات امنیتی و تغییر رمز.
*   **Analytics Service:** مسئول محاسبه آمارهای کاربر (که در بالای صفحه نمایش داده می‌شود).
