# PBank Section: Services (خدمات)
> **Layout:** Hero Slider -> Services Grid
> **Style:** Crypto DApp Browser Style

این بخش بازاری از خدمات کاربردی است که به مرور تکمیل می‌شود.

## 1. اسلایدر ویژه (Featured Slider)
*   **موقعیت:** بالای صفحه.
*   **کاربرد:** برجسته کردن یک خدمت خاص.
*   **مثال:** "تخفیف ۲۰٪ روی بلیط کابل-دبی" یا "پرداخت آنی برشنا".

## 2. لیست خدمات (Services Browser)
لیست آیکون‌های خدمات (مثل اپ‌های داخل یک پوشه):

### فاز اول (نسخه ۱.۰)
1.  **📲 خرید شارژ (Mobile Top-up):**
    *   پشتیبانی از ۵ اپراتور (Roshan, Etisalat, MTN, AWCC, Salaam).
    *   خرید مستقیم.

### فازهای آینده (Coming Soon)
2.  **⚡ برق برشنا (Electricity Bill):** پرداخت قبض DABS.
3.  **💸 مالیات (Tax):** پرداخت مالیات اصناف/خودرو.
4.  **✈️ تکت طیاره (Flights):** رزرو پروازهای داخلی/خارجی.
5.  **🏨 رزرو هتل (Hotels):** رزرو اقامتگاه.

## 3. معماری (Microservices Consideration)
*   **Utility Service:** هسته مرکزی اتصال به APIهای بیرونی (مثل API روشن یا برشنا).
*   **Catalog Service:** لیست خدمات و آیکون‌های آنها را دینامیک نگه می‌دارد (تا بدون آپدیت اپ، خدمت جدید اضافه کنیم).
