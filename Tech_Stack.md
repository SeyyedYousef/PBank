# Tech Stack
> **Approach:** Practical, Fast, Scalable.

## زبان‌ها و فریم‌ورک‌ها
*   **موبایل (Frontend):** React Native (Expo) - برای توسعه سریع و بیلد آسان.
*   **وب (Admin/Agent):** React.js + Vite.
*   **بکند (Backend):** Node.js (NestJS یا Express) - معماری میکروسرویس.
*   **دیتابیس:** PostgreSQL - دیتابیس رابطه‌ای مطمئن برای تراکنش‌های مالی.

## ابزارهای کلیدی
*   **KYC / AI:** اتصال به APIهای گوگل (Vision) یا آمازون (Rekognition) از طریق بکند.
*   **Maps:** استفاده از Google Maps SDK یا Mapbox برای پیدا کردن نمایندگان.

## قوانین توسعه
*   **سادگی:** از کتابخانه‌های استاندارد و پایدار استفاده کنید. نیازی به بازنویسی چرخ نیست.
*   **پرفورمنس:** اپلیکیشن باید روی گوشی‌های میان‌رده هم روان اجرا شود (لیست‌ها با FlatList بهینه شوند).
